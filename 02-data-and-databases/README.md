# Data and Databases

Understanding how modern systems store, retrieve, and protect information.

This module explores database internals, indexing mechanisms, transaction management, query optimization, and storage architecture.

## Why This Matters

Almost every software system depends on databases. Understanding how data is stored and retrieved allows engineers to design efficient applications and troubleshoot performance bottlenecks.

## Topics

* B+ Tree Indexing
* Query Optimization
* ACID Transactions
* Database Locking
* Isolation Levels
* Deadlocks
* Storage Engines

## Learning Goal

Learn how databases make engineering tradeoffs between performance, consistency, reliability, and scalability.
