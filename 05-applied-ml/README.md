# Applied Machine Learning

Understanding machine learning from both mathematical and engineering perspectives.

This module focuses on learning algorithms, model training, feature engineering, and production-ready ML workflows.

## Why This Matters

Machine learning systems are only valuable when they can be trained, evaluated, deployed, and maintained effectively.

## Topics

* Gradient Descent
* Feature Engineering
* Data Leakage
* Model Evaluation
* Training Pipelines
* Production ML Systems

## Learning Goal

Build a strong foundation in both the theory and practical implementation of machine learning systems.
